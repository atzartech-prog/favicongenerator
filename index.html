Cara Pasang di WordPress (Tanpa Kode):
  1. Unduh berkas 192x192 PNG di atas.
  2. Buka dasbor WordPress Anda -> Tampilan (Appearance) -> Sesuaikan (Customize).
  3. Buka tab **Identitas Situs (Site Identity)**.
  4. Di bagian bawah pada **Ikon Situs (Site Icon)**, klik **Pilih Ikon** dan unggah berkas PNG yang baru saja Anda unduh.
  5. Klik **Terbitkan (Publish)**. Selesai!
Cara Pasang di Blogger / Blogspot:
  1. Unduh berkas 32x32 PNG di atas.
  2. Masuk ke dasbor Blogger Anda di blogger.com.
  3. Klik menu **Setelan (Settings)** di sebelah kiri.
  4. Di bawah bagian dasar, klik tulisan **Favicon**.
  5. Klik **Pilih File** lalu unggah berkas PNG yang telah diunduh tadi, lalu klik **Simpan**.

Simpan file favicon-32x32.png di folder root proyek web Anda (satu folder dengan index.html), lalu tempelkan tag HTML berikut di dalam tag <head> halaman HTML Anda:

<link rel="icon" type="image/png" sizes="32x32" href="favicon-32x32.png">
<link rel="apple-touch-icon" href="favicon-192x192.png">

Cara Instan Menggunakan Emoji SVG (Tanpa Perlu Upload File Gambar):

Salin tag HTML tunggal di bawah ini dan tempelkan di dalam tag <head> halaman web Anda. Ikon akan dibaca oleh browser modern sebagai gambar vektor SVG dinamis!

<link rel="icon" href="data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><text y='0.9em' font-size='90'>🚀</text></svg>">

*Catatan: Sangat praktis karena tidak perlu mengunduh/mengunggah file ikon. Didukung penuh oleh Google Chrome, Firefox, Edge, dan Opera terbaru.
