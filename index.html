Cara Pemasangan & Integrasi
Cara Resmi & Paling Mudah (Melalui Menu Kustomisasi):
  1. Unduh berkas PWA Icon (192x192) atau apple-touch-icon di atas.
  2. Masuk ke dasbor WordPress Anda (wp-admin).
  3. Buka menu Appearance (Tampilan) -> Customize (Sesuaikan).
  4. Klik pada tab Site Identity (Identitas Situs).
  5. Di bagian Site Icon, klik **Select Site Icon** dan unggah berkas PNG yang sudah Anda unduh.
  6. Klik **Publish** di kanan atas. WordPress akan menyebarkannya otomatis di seluruh website.
Cara Resmi Pemasangan di Blogger:
  1. Unduh berkas 32x32 PNG atau favicon-16x16 dari generator ini.
  2. Masuk ke dasbor Blogger Anda di blogger.com.
  3. Pilih blog Anda, lalu klik menu **Setelan (Settings)** di kolom kiri.
  4. Di bagian setelan dasar, klik tulisan **Favicon**.
  5. Klik tombol **Pilih File (Choose File)** dan unggah berkas gambar PNG tersebut (pastikan ukuran berkas di bawah 100 KB).
  6. Klik **Simpan (Save)**. Browser mungkin butuh waktu beberapa jam untuk mereset cache dan memunculkan favicon baru.

Ekstrak berkas .ZIP yang Anda unduh dari generator ini dan letakkan di folder root repositori proyek HTML Anda (misal di samping berkas index.html). Kemudian tempelkan kode berikut di dalam tag <head> halaman HTML Anda:

HTML 
<!-- Favicon Utama untuk Browser Umum -->
<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
<link rel="icon" type="image/png" sizes="16x16" href="favicon-16x16.png">
<link rel="icon" type="image/png" sizes="32x32" href="favicon-32x32.png">
<link rel="icon" type="image/png" sizes="48x48" href="favicon-48x48.png">

<!-- Favicon Khusus Apple Touch (iOS Device) -->
<link rel="apple-touch-icon" sizes="180x180" href="apple-touch-icon.png">

<!-- Manifest Khusus Android & PWA -->
<link rel="manifest" href="site.webmanifest">

Untuk mendukung Progressive Web App (PWA), simpan kode JSON manifest berikut di dalam berkas bernama site.webmanifest di root direktori Anda:

JSON 
{
  "name": "My Web App",
  "short_name": "App",
  "icons": [
    {
      "src": "/android-chrome-192x192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "/android-chrome-512x512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ],
  "theme_color": "#3B82F6",
  "background_color": "#ffffff",
  "display": "standalone"
}
